Configuration

  • Install: hr_attendance_ctrl
  • Go to Attendances → Office Locations
  • Create an office and define the polygon boundary (WKT / map editor, depending on your setup)
  • Employees can use the standard Check-In / Check-Out flow

    • Note: GPS accuracy depends on device/browser permissions and environment. Encourage employees to enable location permission.

    HR Attendance Configuration and Controls

    Configure and manage attendance rules with geo-zone based access control. Define allowed office areas and enforce location-aware check-in and check-out policies.

    Set up office locations using GPS coordinates or geo-zones. Each location can be configured to validate employee presence during check-in and check-out.
