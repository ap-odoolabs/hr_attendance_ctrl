# HR Attendance Geo-Control

Advanced Geolocation Attendance for Odoo 18  
Developed by: **Adi Pramono (AP Odoo Labs)**

## Features

- GPS-based check-in/check-out
- Multi-office geofence polygon
- PostGIS geometry validation
- Real-time map UI using Leaflet
- Admin map view
- Anti-GPS spoofing logic

## Requirements

- PostgreSQL with PostGIS extension
- Odoo 18 Community or Enterprise
- HTTPS recommended for precise GPS
